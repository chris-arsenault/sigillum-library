# DSL Card Index

These files are auditionable production-DSL technique card specimens.
Each card is a standalone `production_piece` whose phrases use the local
surface that best exposes the musical job: degrees, intervals, or split
pitch/rhythm. The notes are for auditioning and understanding the card,
not for copying into a later score as final material.
