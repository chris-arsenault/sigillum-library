# craft/ — the craft annexes (research, verbatim)

The hard-won craft the stages bind to. Not ordered — read whichever the task needs (a card's `cite`
points here; the procedures name them).

**Texture & orchestration craft**
- `texture_craft.md` — the texture-craft overview (where the texture layer sits in the per-section flow).
- `ornamentation.md` — ornament recipes, fingerprints, placement checklist, density caps.
- `orchestral_rhythm.md` — figure catalogue, the complementary-rhythm law, strata, cross-rhythm.
- `chord_scoring.md` — spacing / low-interval limits, doubling, voicing templates, harmonic-rhythm law.
- `dialogue_doubling.md` — antiphony, fills, echoes, relays, mixture recipes, halos.
- `keyboard_figuration.md` — keyboard/figuration standards; the figuration taxonomy; the rhythmic-sentence
  law; high-water-mark + dialect doctrine. (Its internal sections are cited as `s1`…`s6g`.)
- `texture_identity_contrast_ledger.md` — Phase-A identity/contrast ledger: continuity identity,
  section job, fractalization target, detail mode, reserve role, and contrast relation.

**Melody craft**
- `phrasing_variation_line.md` — the composer-decision lessons (repetition budgets,
  pleasant-vs-dissonant subversion, long-note placement, the singable-skeleton test,
  bent runs, elided phrases); cards cite it as `phrasing_variation_line:sN`; specimens
  in `dsl:phrasing` / `dsl:variation` / `dsl:linecraft`.
- `melody_primacy.md` — earworm research + the melody laws (R1–R10).
- `melody_theory_foundations.md` — the theory of building a strong single line (its sections cited as `§N`).
- `melody_progression_exemplars.md` — corpus exemplars of melody → implied progression.
- `exposing_the_progression.md` — how a single line exposes/implies its harmony.
- `MELODY_CRAFT_DOCTRINE.md` — the distilled melody-writing stance (guidance, not law).

**Diagnostic**
- `texture_diagnosis.md` — the texture-crisis evidence + diagnostic (cautionary reference).
