# Card Library Location

The canonical reusable technique library now lives at `technique_library/`.

- `technique_library/dsl/README.md` is the production-DSL source guide for new
  work.
- `technique_library/dsl/cards/` contains the auditionable DSL card specimens.
- `technique_library/INDEX.md` is the card-library spec and workflow.
- `technique_library/16th_note_figures.md` is the prose catalogue for the
  `dsl:figures.16th` cards.

This file remains only as a legacy pointer for old documentation.
